# PackHelper-Ahk

  Is a project that help me pass through the hard day at work. Since some processes are annoying and some keys are also annoying, so I have an idea it would be good if I can bypass this process with something.
So I got to know AutoHotkeys, with this I can compress several process into just one button click.

 # How to use:
- Left (<=), Right (=>) arrow: To select between each package barcode.
- Spacebar: To send input package barcode and close the sale.
- Q: To change packing wave paper.
